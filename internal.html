Internal Parts

CPU

The CPU also known as the proccessor is the most

important component in a computer, without it

the computer will not work. The CPU does all of

the calculations and without the CPU it is not a computer

GPU

The GPU or Graphics proccessing Unit is a

chip that is specialzed for visual functions

if possible the GPU will render all the images

and animations that appear on the monitor

RAM

Ram or random access memory is plugged

directly into the motherboard and commmunicates

with the CPU and the storage. The data put into

random access memory is data that needs to be

retreived very quickly

Hard Drive vs SSD

Hard Drive
  • Stores information on a Physical disc
  • uses a mechanical arm to retreive data
  • large Capacity
  • relatively cheap
SSD
  • Stores information on Micro Chips
  • No moving parts
  • smaller capacity
  • more expensive
